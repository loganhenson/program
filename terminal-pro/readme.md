## Install the font
```console
brew install font-jetbrains-mono-nerd-font
```

## Work on it
> Get rust going
https://www.rust-lang.org/tools/install
rustup default nightly
```console
cd src-tauri
cargo build
```

> Get npm deps (elm)
```console
npm install
```

> Build
```console
npm run watch
```

## Test
```console
npm run test
```

## Build for dmg
```console
npm run build
```
